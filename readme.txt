== Description ==

This plugin will help you Create/Add your Store locations information, You can add as many as you have.

To Create/Add an store location information, goto Store Locations --> add new Locations


== Installation ==

1. Go to your admin area and select Plugins -> Add new from the menu or Browse plugin if you download it already. 
2. Search for "lam-store-locator".
3. Click install.
4. Click activate.


== Frequently Asked Questions ==

= How to display the map to the content? =

You can display using the shortcode [Store-location-map]. 

= How to display the store location list? =

Just copy and paste [Store-location-list]. [Store-location-list list-class="put here your desired class-name"]
